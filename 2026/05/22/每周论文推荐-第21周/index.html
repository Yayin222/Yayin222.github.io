每周论文推荐 - 第21周

发表于2026-05-22|更新于2026-05-22|论文推荐
|浏览量:

本周共收集 17 篇 论文，涵盖 Agent 与 安全 两大方向。Agent 方向持续火热，多篇工作聚焦自主 Agent 的持续学习、工具编排与评估；安全方向关注越狱攻击、后门攻击与 Agent 网络的协议层安全。

🤖 Agent 方向

1. SOLAR: A Self-Optimizing Open-Ended Autonomous Agent for Lifelong Learning and Continual Adaptation

来源: AAAI 2026 (CEUR Workshop Proceedings)
链接: arXiv:2605.20189
核心思想: 提出一个开放式自主 Agent，具备终身学习和持续自适应能力，能够在开放环境中不断优化自身行为策略。

2. COSMO-Agent: Tool-Augmented Agent for Closed-loop Optimization, Simulation, and Modeling Orchestration

来源: arXiv
链接: arXiv:2605.20190
核心思想: 工具增强型 Agent，专注于闭环优化、仿真与建模编排，实现科学计算场景中的自动化工作流。

3. OSCToM: RL-Guided Adversarial Generation for High-Order Theory of Mind

来源: arXiv
链接: arXiv:2605.20423
核心思想: 利用强化学习引导的对抗生成方法，提升 Agent 的高阶心智理论（Theory of Mind）推理能力。

4. AgentCo-op: Retrieval-Based Synthesis of Interoperable Multi-Agent Workflows

来源: arXiv
链接: arXiv:2605.20425
核心思想: 基于检索的多 Agent 工作流合成框架，实现异构 Agent 之间的互操作与协同编排。

5. Open-World Evaluations for Measuring Frontier AI Capabilities

来源: arXiv
链接: arXiv:2605.20520
核心思想: 提出开放世界评估范式，用于衡量前沿 AI 系统在真实复杂环境中的综合能力。

6. AgentAtlas: Beyond Outcome Leaderboards for LLM Agents

来源: arXiv
链接: arXiv:2605.20530
核心思想: 超越传统结果排行榜，构建 LLM Agent 的多维度能力图谱，提供更细粒度的评估视角。

7. COAgents: Multi-Agent Framework to Learn and Navigate Routing Problems Search Space

来源: LION 2026
链接: arXiv:2605.20618
核心思想: 多 Agent 框架，用于学习和导航路径规划问题的搜索空间，结合多智能体协作提升求解效率。

8. HANA: Hierarchical Agent-native Network Architecture

来源: IEEE Networking Letters
链接: arXiv:2605.20608
核心思想: 提出分层式 Agent 原生网络架构，将 AI Agent 深度融入网络设计与运行中。

9. GROW: Aligning GRPO with State-Action Modeling for Open-World VLM Agents

来源: arXiv
链接: arXiv:2605.20246
核心思想: 将 GRPO 算法与状态-动作建模对齐，用于训练开放世界中的视觉语言模型（VLM）Agent。

10. Multi-Agent Reinforcement Learning for Safe Autonomous Driving Under Pedestrian Behavioral Uncertainty

来源: arXiv
链接: arXiv:2605.20255
核心思想: 针对行人行为不确定性，提出多智能体强化学习框架以实现安全自动驾驶。

🔒 安全方向

11. Adaptive Probe-based Steering for Robust LLM Jailbreaking

来源: ICML 2026
链接: arXiv:2605.20286
核心思想: 提出自适应探针引导方法，实现对 LLM 的鲁棒越狱攻击，揭示了当前安全对齐的薄弱环节。

12. Pramana: A Protocol-Layer Treatment of Claim Verification in Autonomous Agent Networks

来源: arXiv
链接: arXiv:2605.20312
核心思想: 从协议层出发，为自主 Agent 网络中的声明验证提供形式化处理方案，保障多 Agent 通信安全。

13. Refusal Evaluation in Coding LLMs and Code Agents: A Systematic Review

来源: arXiv
链接: arXiv:2605.20351
核心思想: 系统综述编码 LLM 和代码 Agent 中的拒绝评估机制，分析安全拒绝策略的有效性与局限性。

14. SELFCI: Complementary Self-Distillation for Contextual Integrity in LLMs

来源: arXiv
链接: arXiv:2605.20258
核心思想: 通过互补自蒸馏方法维护 LLM 的上下文完整性，防止上下文泄露与信息污染。

15. Trusted Weights, Treacherous Optimizations? Optimization-Triggered Backdoor Attacks on LLMs

来源: arXiv
链接: arXiv:2605.20641
核心思想: 揭示一种新型后门攻击——优化触发后门，即使权重来源可信，优化过程本身也可能植入后门。

16. HBHC: Heartbeat-Bound Hierarchical Credentials — Cryptographic Revocation for AI Agent Swarms

来源: arXiv
链接: arXiv:2605.20704
核心思想: 提出心跳绑定分层凭证机制，为 AI Agent 群体提供密码学级别的凭证撤销能力。

17. Geometry-Lite: Interpretable Safety Probing via Layer-Wise Margin Geometry

来源: arXiv
链接: arXiv:2605.20241
核心思想: 通过逐层边界几何分析，实现可解释的安全探测，为 LLM 安全性评估提供新视角。
