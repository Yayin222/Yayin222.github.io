每周论文推荐 - 第19周

发表于2026-05-08|更新于2026-05-08|论文推荐
|浏览量:

本周精选论文

本周共收录 15 篇新论文，涵盖 Agent 推理、LLM 安全、漏洞挖掘、供应链安全等多个方向。

1. LCM: Lossless Context Management

来源: arXiv
链接: arxiv.org/abs/2605.04050
核心思想: 提出无损上下文管理方案，优化 LLM Agent 的长期记忆与上下文窗口利用，解决长对话中信息丢失问题。

2. Strat-Reasoner: Reinforcing Strategic Reasoning of LLMs in Multi-Agent Games

来源: arXiv
链接: arxiv.org/abs/2605.04906
核心思想: 通过强化学习增强 LLM 在多智能体博弈中的战略推理能力，提升复杂对抗场景下的决策质量。

3. On the (In-)Security of the Shuffling Defense in Transformer Secure Inference

来源: arXiv (ACL 2026)
链接: arxiv.org/abs/2605.04901
核心思想: 揭示 Transformer 安全推理中 shuffle 防御机制的安全缺陷，证明该防御在特定条件下可被绕过。

4. Sparse Backdoor: Undetectable Backdoors in Model Parameters

来源: arXiv
链接: arxiv.org/abs/2605.04209
核心思想: 提出稀疏后门攻击方法，在模型参数中植入难以检测的后门，对 AI 供应链安全构成威胁。

5. AFL-ICP: Specification-Guided Fuzzing for Industrial Control Protocols

来源: arXiv
链接: arxiv.org/abs/2605.04760
核心思想: 结合 LLM 辅助的规范引导模糊测试框架，针对工业控制协议进行高效漏洞挖掘。

6. SQSD: Quantifying Sample-Level Safety Degradation in LLM Fine-tuning

来源: arXiv (ICML 2026)
链接: arxiv.org/abs/2605.04572
核心思想: 量化微调过程中样本级别的安全退化程度，为 LLM 安全微调提供度量标准。

7. NeWTral: Automatic Safety Alignment Restoration through Neural Weight Translation

来源: arXiv
链接: arxiv.org/abs/2605.04992
核心思想: 通过神经权重翻译自动恢复被 LoRA 微调破坏的安全对齐，提供新的防御思路。

8. Sparse Tokens Suffice: Jailbreaking Audio Language Models via Token-Aware Gradient Optimization

来源: arXiv
链接: arxiv.org/abs/2605.04700
核心思想: 利用稀疏 token 和梯度优化实现音频语言模型的越狱攻击，揭示音频模态的安全风险。

9. Laundering AI Authority with Adversarial Examples

来源: arXiv
链接: arxiv.org/abs/2605.04261
核心思想: 利用对抗样本”洗白”AI 权威性，操纵视觉语言模型生成误导性信息。

10. Misrouter: Exploiting Routing Mechanisms for Input-Only Attacks on Mixture-of-Experts LLMs

来源: arXiv
链接: arxiv.org/abs/2605.04446
核心思想: 利用 MoE 模型的路由机制进行纯输入攻击，绕过安全防护实现安全规避。

11. Kumushi: Root-Cause-Driven Automated Vulnerability Repair

来源: arXiv
链接: arxiv.org/abs/2605.04251
核心思想: 基于根因分析的自动化漏洞修复框架，结合 LLM Agent 实现精准的故障定位与修复。

12. Pen-Strategist: A Reasoning Framework for Penetration Testing Strategy Formation and Analysis

来源: arXiv
链接: arxiv.org/abs/2605.04499
核心思想: 基于 Qwen 的渗透测试推理框架，辅助安全人员制定和分析攻击策略。

13. AgentTrust: Runtime Safety Evaluation and Interception for AI Agent Tool Use

来源: arXiv
链接: arxiv.org/abs/2605.04785
核心思想: 提出 Agent 工具调用的运行时安全评估与拦截机制，适用于 MCP 等工具调用场景。

14. SLYP: Agentic Vulnerability Reasoning on Windows COM Binaries

来源: arXiv
链接: arxiv.org/abs/2605.05000
核心思想: 利用 Agent 流水线对 Windows COM 二进制文件进行自动化漏洞推理与利用分析。

15. Measuring Compositional Vulnerability Induction in Coding Agents (MOSAIC-Bench)

来源: arXiv
链接: arxiv.org/abs/2605.03952
核心思想: 提出 MOSAIC-Bench 基准，衡量编码 Agent 引入组合型漏洞的能力，揭示 AI 代码生成的安全风险。

本周小结

本周论文呈现几个明显趋势：

  1. Agent 安全成为焦点 — AgentTrust、ARGUS、SLYP 等多篇论文关注 Agent 工具调用、提示注入、漏洞推理的安全问题
  2. 攻防对抗持续升级 — MoE 路由攻击、音频越狱、稀疏后门等新型攻击手法不断涌现
  3. 自动化漏洞挖掘成熟化 — Kumushi、AFL-ICP、Pen-Strategist 展示了 LLM 辅助安全研究的实际落地

本周论文均已在本地归档，欢迎交流讨论。
